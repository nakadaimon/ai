# harness

> How the work gets done. Tools, verbs, and automatic triggers.

1. `operations.md` — tools the model has access to and review protocols
2. `commands.md` — triggers the user invokes (`ingest`, `lint`)
3. `hooks.md` — triggers the agent fires automatically when an observable state is detected

All files start small and stay small. Heavier patterns live as operational extensions in `README.md` — adopt, adapt, or ignore.
