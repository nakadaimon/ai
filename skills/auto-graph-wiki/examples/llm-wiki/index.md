# Example LLM-Wiki — Index

> An example wiki demonstrating how `auto-graph-wiki` works. Replace this with your own wiki content.
> Pages live under `pages/`, each with frontmatter that the indexer reads.
> Run `python ../../scripts/reindex.py` to build the search index.

This wiki contains four example pages across two domains: `cooking` and `productivity`. They are intentionally generic — the goal is to show what frontmatter looks like and how it powers cheap retrieval, not to teach any particular subject.
